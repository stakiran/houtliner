==================
    houtliner
==================

軽くてパワフルなアウトライナーを、秀丸エディタで。


■ファイルについて
以下は本体ファイルです。

・outline.hilight         強調定義ファイル（色付けやアウトラインの認識）
・houtliner.mac           各種操作をメニュー形式で提供するマクロ
・create_zoomfile.bat     ズーム機能時に呼び出される補助スクリプト

以下はドキュメントです。

・readme.txt              このファイル
・README.md               詳しい操作方法などを記したマニュアル
・changelog.md            更新履歴
・LICENSE                 本ソフトウェアのライセンスを記したテキスト

以下はサンプルのアウトラインファイルです。

・sample.outline          houtliner の機能を一通り使ったサンプル
